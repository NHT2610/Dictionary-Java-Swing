HƯỚNG DẪN BIÊN DỊCH CHẠY MÃ NGUỒN:
- Môi trường Visual Studio Code, đã cài Extension Pack for Java.
- JDK: phiên bản JDK mà ứng dụng sử dụng là JDK19
- Nhấn Ctr + F5 để chạy chương trình
** Nếu bạn muốn chạy file 20120608.jar:
B1: Trên terminal, điều hướng đến thư mục 20120608 (thư mục dự án)
B2: Gõ lệnh: java --enable-preview -jar release/20120608.jar (lưu ý là --enable-preview chỉ hỗ trợ Java 12 trở lên)

CẤU TRÚC THƯ MỤC:
20120608:
	src: tương đương thư mục Source, chứa source code của chương trình
	release: tương đương thư mục Release, chứa tập tin 20120608.jar của chương trình
	lib: tương đương thư mục Libs, chứa các thư viện
	Readme.txt: tập tin đính kèm thông tin sinh viên, mô tả cấu trúc thư mục, hướng dẫn chạy chương trình.

	Ngoài ra còn một số thư mục khác:
	res: chứa các tài nguyên sử dụng cho ứng dụng
	bin: chứa các file .class được biên dịch
	.vscode: thư mục chứa tập tin cấu hình của ứng dụng
